# Code of Conduct ## Our Pledge We pledge to make participation in our project respectful and inclusive for everyone. ## Our Standards - Use respectful language - Be supportive of others - Accept constructive criticism ## Enforcement Responsibilities Project maintainers are responsible for enforcing the code of conduct. ## Scope This code of conduct applies within all project spaces. ## Enforcement Instances of abusive behavior may be reported to the maintainers. ## Enforcement Guidelines Warnings may be issued for unacceptable behavior. ## Attribution This Code of Conduct is adapted from the Contributor Covenant.
